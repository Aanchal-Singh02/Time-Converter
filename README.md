# Time Converter

A simple web application that allows you to convert time between 12-hour and 24-hour formats.

## Features

- Converts time from 12-hour format (e.g., 02:30 PM) to 24-hour format (e.g., 14:30)
- Converts time from 24-hour format (e.g., 14:30) to 12-hour format (e.g., 02:30 PM)
- Provides clear and concise output
- User-friendly interface with a single input field and a convert button
- Enter Key Support

## How to Use

1. Open the `index.html` file in your web browser.
2. Enter the time you want to convert in the input field.
    - For 12-hour format, use the format: `HH:MM AM/PM` (e.g., 02:30 PM)
    - For 24-hour format, use the format: `HH:MM` (e.g., 14:30)
3. Click the "Convert" button or press Enter key.
4. The converted time will be displayed below the input field.

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

- `index.html`: Contains the HTML structure of the web page.
- `style.css`: Contains the CSS styles for the web page.
- `script.js`: Contains the JavaScript code for time conversion logic.
